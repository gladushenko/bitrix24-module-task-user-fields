# Changelog

## [1.1.0] - 2026-07-06

### Added

- Добавлена привязка области пользовательских полей к проектам Bitrix24.
- Добавлен вывод блока пользовательских полей только для задач выбранных проектов.

### Changed

- Сохранение значений пользовательских полей в карточке задач переведено с использованием API битрикса, чтобы при сохранении корректно вызывались обработчики событий задач.

## [1.0.0]

### Added

- Добавлен модуль `gladushenko.taskuserfields` для Bitrix24.
- Добавлена страница настроек в административной панели.
- Добавлен выбор пользовательских полей сущности `TASKS_TASK` для отображения в карточке задачи.
- Добавлена настройка заголовка блока пользовательских полей.
- Добавлена возможность задавать пользовательские подписи полей в карточке задачи.
- Добавлена возможность скрывать стандартный блок пользовательских полей Bitrix24.
- Добавлен вывод выбранных пользовательских полей в полной карточке задачи.
- Добавлено редактирование значений пользовательских полей прямо из карточки задачи.
- Добавлена поддержка типов `integer`, `double`, `string`, `enumeration`, `boolean`, `datetime`, `date`, `money`, `url`, `iblock_element`, `iblock_section`.
- Добавлен AJAX-контроллер для получения и сохранения значений пользовательских полей задачи.
- Добавлены frontend-скрипты для встраивания блока в карточку задачи Bitrix24.
- Добавлены уведомления с понятным текстом ошибки при проблемах сохранения задачи.
- Добавлен README с описанием установки, настройки и структуры проекта.
